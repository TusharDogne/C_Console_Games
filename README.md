# 🎮 C Console Games Collection

## 📌 Overview

This repository contains a collection of console-based games developed in C.
Each project focuses on core programming concepts such as loops, condition handling, arrays, and user interaction.

The goal of this collection is to strengthen logic building and create interactive terminal-based applications.

---

## 🎯 Games Included

### 🐍 Snake and Ladder

* Two-player console game
* Random dice generation
* Board rendering with snakes and ladders
* Position tracking and win conditions

---

### 🔢 Number Guessing Game

* Random number generation
* Limited attempts
* Hint system (higher/lower)

---

### ❌⭕ Tic Tac Toe

* Two-player turn-based game
* 3x3 grid system
* Win and draw detection

---

### 🧠 Hangman

* Word guessing game
* Limited attempts
* Letter tracking and display

---

### 📚 Quiz Game

* Multiple choice questions
* Score tracking system
* Final performance evaluation

---

## ⚙️ Technologies Used

* Language: C
* Libraries: stdio.h, stdlib.h, time.h

---

## ▶️ How to Run

### Compile:

```bash
gcc filename.c -o game
```

### Run:

```bash
./game
```
